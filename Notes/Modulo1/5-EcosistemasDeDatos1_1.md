## Procesos de Análisis de Datos
Todos los procesos de Análisis de Datos pueden variar según la empresa específica de la cual se esté hablando, por lo cual mencionaré procesos característicos y evaluaremos que, aunque sean diferentes fases, llegan a concluir en un mismo resultado y a sí mismo llegan a ser similares a los otros como generalidad.

---

### Proceso que constituye el Certificado Google Analytics
1. Plantear: reto, objetivo o pregunta empresarial  
2. Preparar: generación, recopilación, almacenamiento y gestión de datos  
3. Procesar: limpieza de datos e integridad de los datos  
4. Analizar: exploración, visualización y análisis de datos  
5. Compartir: comunicar e interpretar los resultados  
6. Actuar: poner en práctica los conocimientos para resolver el problema  

---

### Proceso de análisis de datos de EMC (Dell)
1. Descubrimiento  
2. Preprocesamiento de datos  
3. Planificación de modelos  
4. Construcción del modelo  
5. Comunicación de resultados

### Proceso de Analítica de Datos basado en proyectos

### Proceso de Analítica de Datos Big Data
Thomas Erl, Wajid Khattak y Paul Buhler desarrollaron este proceso de analítica de datos con orientación a **Big Data**, el cual se descompone en pasos más pequeños y específicos, pero no le añade complejidad, sino mayor nivel de detalle. Este proceso se desarrolló en el libro **Big Data Fundamentals: Concepts, Drivers & Techniques**.

1. Evaluación del caso de negocio  
2. Identificación de datos  
3. Adquisición y filtrado de datos  
4. Extracción de datos  
5. Validación y limpieza de datos  
6. Agregación de datos y representación  
7. Análisis de datos  
8. Visualización de datos  
9. Utilización de los resultados del análisis  

---

## Conclusión
A pesar de que existen múltiples metodologías y enfoques para el análisis de datos, todas comparten un objetivo común: transformar los datos en información útil que permita tomar mejores decisiones. Las diferencias entre los procesos responden al contexto, la escala y las necesidades del proyecto, pero en esencia mantienen una estructura similar orientada a generar valor a partir de los datos.
